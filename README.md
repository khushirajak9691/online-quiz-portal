
# Online Quiz Portal

Frontend-only project using *HTML, CSS, and JavaScript, implementing **MVC architecture* for better code organization and maintainability.

## Project Overview
This project is an *Online Quiz Portal* where users can take quizzes, track their progress, and see their scores. It demonstrates basic *frontend development skills* along with understanding of software architecture concepts.

## Features
- *MVC Architecture*: Model (quiz data), View (HTML/CSS), Controller (JS logic)
- *Dynamic Quiz Questions*: Users can answer multiple-choice questions
- *Timer for Each Question*: 15-second countdown for better UX
- *Progress Tracking*: Shows current question out of total
- *Score Calculation*: Real-time score display
- *High Score Storage*: Saves best score using LocalStorage
- *Restart Quiz Functionality*: Users can retake the quiz anytime
- *Responsive Design*: Works on desktop and mobile screens
